## Course Overview

This companion track provides practical depth and exam readiness for certification-level performance.

- Respond accurately to common IRS notices
- File amended returns with proper audit trail
- Apply baseline tax planning logic to real scenarios
- Pass both written and practical final assessments

## Prerequisites

- Completion of core classes 1–22 is strongly recommended

## Delivery Notes

- Delivered as bonus content plus assessment workshops
- Includes final readiness and remediation sessions

## Grading

- Bonus module completion: Pass/Fail
- Final Assessment A (written): 40%
- Final Assessment B (practical): 60%
